# DOS Walkthroughs

This directory contains walkthrough material for understanding and using the canonical DOS package.

## Recommended Walkthrough
1. Read the root `README.md`.
2. Read `docs/INSTANCE_CREATION.md`.
3. Inspect `dos/instance-template/` to see what a downstream instance receives.
4. Compare that with `dos/reference/example-instance/` to understand the target quality bar.
