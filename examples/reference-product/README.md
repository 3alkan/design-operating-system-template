# Reference Product Example

This example shows a completed instance of the template for a work intake and approval platform.

## Product Summary
- Product: Flow Approval Hub
- Purpose: capture internal work requests, route them through policy-based approval, issue fulfillment tasks, and keep a durable audit trail and operator view.
- Audience: developers and LLMs learning what a completed, implementation-ready, stack-independent design package looks like.

## Validation
- `python scripts/validate_template.py --mode instance --root examples/reference-product`

## Why This Example Exists
- Demonstrates the quality bar for a completed instance.
- Shows how goals, scenarios, capabilities, contracts, architecture, ADRs, quality, and operations connect.
- Serves as a benchmark for repo validation and future template changes.
