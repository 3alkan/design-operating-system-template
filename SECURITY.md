# Security Policy

## Scope
This is a template repository. Security issues may include:
- Exposed secrets or tokens in docs/templates/examples.
- Unsafe guidance that could cause insecure defaults in downstream projects.
- Vulnerabilities in repository automation/workflows (if present).

## How To Report
- Preferred: GitHub Security Advisories for this repository.
- Alternative: email [PROJECT_CONTACT_EMAIL].

Please include reproduction steps, affected files, and impact.

## Response Expectations
- Initial acknowledgment: best effort within 5 business days.
- Follow-up and triage: best effort, depending on severity and maintainer availability.
- Fix communication: via advisory, issue, or release notes as appropriate.

## Non-Security Issues
For general bugs or improvements, use regular GitHub issues.
