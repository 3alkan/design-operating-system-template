# Diagram Policy

## Rules
- Mermaid-only diagrams.
- Diagrams must be embedded in Markdown fenced blocks using ` ```mermaid `.
- GitHub renders Mermaid directly in Markdown.
- No committed PNG/SVG images required.
- Keep diagrams small and split by concern.
- Avoid fancy styling; optimize for readability and maintenance.

## Allowed Diagram Types
- `flowchart`
- `sequenceDiagram`
- `stateDiagram-v2`
